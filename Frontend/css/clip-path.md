# CSS: clip-path

## Что я узнал

Свойство `clip-path` позволяет обрезать элементы по различным формам. В теории обычно рассматриваются простые фигуры: `circle()`, `ellipse()`, `polygon()`. На практике вы используете более сложные конструкции: `clip-path: path('M...')` для рисования кривыми Безье и привязку к SVG-маскам через `url(#shape)`. Это позволяет создавать сложную векторную графику прямо в CSS и интегрировать её с SVG. Для работы с такими clip-path важно понимать синтаксис `path()`, команды движения по контуру (M, L, C и др.) и логику координат.

Таким образом, вы переходите от простых геометрических обрезок к полноценной векторной графике, что расширяет возможности дизайна, но требует изучения новых правил синтаксиса.

### Пример
```css
/* Простая фигура */
.element {
  clip-path: circle(50% at 50% 50%);
}

/* Кривая Безье через path */
.element-complex {
  clip-path: path('M10 80 C 40 10, 65 10, 95 80 S 150 150, 180 80');
}

/* Привязка к SVG-маске */
.element-svg {
  clip-path: url(#my-shape);
}
```
