# CSS функции: calc(), min(), max(), clamp()

## Что я узнал

В CSS существуют специальные **математические функции**, которые позволяют вычислять значения прямо внутри стилей. Это особенно полезно для **адаптивной верстки**, когда размеры элементов должны зависеть от размера экрана, контейнера или других значений.

Самая известная функция — **`calc()`**. Она позволяет выполнять математические операции прямо в CSS.

Поддерживаемые операции:

- `+` — сложение  
- `-` — вычитание  
- `*` — умножение  
- `/` — деление  

Главная особенность `calc()` — возможность **комбинировать разные единицы измерения**, например `px`, `%`, `vw`, `vh`.

Пример:

```css
width: calc(100% - 30px);
```

Это означает: ширина элемента будет равна ширине родителя минус 30 пикселей.

### Правила использования calc()

1. Между операторами `+` и `-` обязательно должны быть **пробелы**.
2. Умножение и деление работают только если **один из операндов — число**.
3. Нельзя выполнять операции между **несовместимыми единицами** (например `% * px`).

Поэтому выражения вроде:

```css
max-width: calc(75% * 100px);
```

будут **некорректными**, потому что браузер не может перемножить разные единицы.

### Связка calc() и CSS-переменных

`calc()` отлично работает вместе с **CSS переменными (`var()`)**, что делает систему размеров очень гибкой.

Например можно задать параметры контейнера:

```css
:root {
  --container-max-width: 1170px;
  --container-padding-x: 15px;
}
```

А затем вычислить реальную ширину:

```css
max-width: calc(var(--container-max-width) + var(--container-padding-x) * 2);
```

### Адаптивная типографика

`calc()` часто используют для **плавного изменения размера шрифта** в зависимости от ширины экрана.

Пример:

```css
font-size: calc(16px + (36 - 16) * ((100vw - 360px) / (1920 - 360)));
```

Это позволяет плавно увеличивать размер текста между двумя разрешениями экрана.

### min(), max() и clamp()

Помимо `calc()` существуют функции управления ограничениями размеров.

#### min()

Выбирает **наименьшее значение** из списка.

```css
width: min(200px, 100% / 3);
```

Элемент будет не больше 200px или одной трети ширины контейнера.

#### max()

Выбирает **наибольшее значение**.

```css
width: max(250px, 50%);
```

Элемент будет минимум 250px, но может увеличиваться до 50%.

#### clamp()

Самая удобная функция — задаёт **минимум, предпочтительное значение и максимум**.

```css
font-size: clamp(24px, 5vw, 60px);
```

Это означает:

* минимум — 24px
* желательно — 5vw
* максимум — 60px

Часто используется для **адаптивной типографики и блоков интерфейса**.

### Пример

```javascript
.box {
  /* Корректные формулы */
  width: calc(10px + 20px);
  height: calc(5% + 3% - 1%);
  margin: calc(10px * 3);
  padding: calc(30vw / 5);

  font-size: calc(16px + (36 - 16) * ((100vw - 360px) / (1920 - 360)));

  line-height: calc(1.5 / 0.85);
  transition-duration: calc(1s * 3);
  rotate: calc(360deg / 4);
  translate: calc(100vw / (3 * 2));
  opacity: calc(1 - 0.3);
  z-index: calc(10 + 1);

  /* Некорректные примеры */
  max-width: calc(75% * 100px);
  max-height: calc(100vh / 30px);
}

:root {
  --container-max-width: 1170px;
  --container-padding-x: 15px;
}

.container {
  max-width: calc(var(--container-max-width) + var(--container-padding-x) * 2);
  margin-inline: auto;
  padding-inline: var(--container-padding-x);
}

.catalog-list {
  display: flex;
  flex-wrap: wrap;
}

.catalog-item {
  width: calc(100% / 3);
}

.box1 {
  width: min(200px, 100% / 3);
}

.box2 {
  width: max(250px, 50%);
}

.menu {
  width: clamp(200px, 30vw, 400px);
}

.title {
  font-size: clamp(24px, 5vw, 60px);
}
```
