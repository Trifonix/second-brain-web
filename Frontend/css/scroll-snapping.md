# CSS: Scroll Snapping

## Что я узнал

Scroll Snapping — это CSS-технология, которая позволяет «зафиксировать» прокрутку на определённых элементах. Она идеально подходит для создания плавных слайдеров, галерей или каруселей без использования JavaScript. Основные свойства:

- `scroll-snap-type`: задаёт направление и обязательность «защелкивания» прокрутки (например, `x mandatory` фиксирует горизонтальную прокрутку на каждом snap-элементе).
- `scroll-snap-align`: определяет, как дочерний элемент «прилипает» к контейнеру при прокрутке (например, `start`, `center` или `end`).

Главное преимущество — улучшение UX без лишних скриптов и полный контроль над поведением прокрутки.

### Пример
```css
/* Контейнер галереи */
.gallery {
  display: flex;
  overflow-x: scroll;
  scroll-snap-type: x mandatory;
}

/* Элементы галереи */
.gallery-item {
  flex: 0 0 80%;
  scroll-snap-align: start;
  margin-right: 20px;
}
```
