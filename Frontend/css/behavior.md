# Управление взаимодействием и поведением элементов

## Что я узнал

CSS предоставляет возможность контролировать, как элементы реагируют на действия пользователя и как отображаются на странице. Свойство `cursor` изменяет вид курсора при наведении на элемент, позволяя указывать, например, `pointer` для кликабельных областей или `text` для текстовых полей. `user-select` управляет возможностью выделения текста пользователем: `none` запрещает выделение, а `auto` оставляет стандартное поведение. Свойство `pointer-events` регулирует, может ли элемент реагировать на мышь и события клика. `resize` задаёт возможность изменения размера текстовых полей или блоков: горизонтально, вертикально или в обоих направлениях. Наконец, `scroll-behavior: smooth` делает прокрутку плавной, улучшая UX при переходах по якорям и скроллинге.

Применение этих свойств важно для создания интерактивных интерфейсов, предотвращения случайных действий пользователя и улучшения визуального взаимодействия.

### Пример
```javascript
/* Курсор и блок */
.box {
  width: 100px;
  height: 100px;
  background-color: lightskyblue;
  cursor: pointer; /* при наведении курсор меняется на руку */
}

/* Блоки с отступами и inline-block */
.div {
  height: 50px;
  background-color: lightgreen;
  display: inline-block;
  margin: 15px 15px 0 0;
  padding: 5px;
}

/* Запрет выделения текста */
.box1, button {
  user-select: none; /* нельзя выделить текст */
}

/* Управление событиями мыши */
.box2 {
  padding: 10px;
  background-color: lightskyblue;
  pointer-events: none; /* блок не реагирует на клики */
}
.btn {
  pointer-events: auto; /* кнопка активна для клика */
}

/* Изменение размеров */
textarea {
  resize: vertical; /* только вертикальное изменение */
}
.box3 {
  width: 90px;
  height: 90px;
  background-color: lightcoral;
  overflow: hidden;
  resize: both; /* можно менять размер в обоих направлениях */
}

/* Плавная прокрутка */
html {
  scroll-behavior: smooth;
}
```
