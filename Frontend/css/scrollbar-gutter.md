# CSS: scrollbar-gutter

## Что я узнал

Свойство `scrollbar-gutter` позволяет управлять пространством, которое занимает полоса прокрутки, предотвращая смещение или «прыжки» контента при её появлении или исчезновении. Особенно полезно для страниц с динамическим контентом, когда прокрутка может появляться в определённых ситуациях. Значение `stable` резервирует место под scrollbar заранее, обеспечивая стабильное отображение элементов страницы.

Это современное CSS-свойство, которое дополняет знания о `overflow` и Box Model, добавляя контроль над визуальной стабильностью интерфейса.

### Пример
```css
html {
  scrollbar-gutter: stable;
}
```
