# Атрибут style и приоритет CSS

## Что я узнал

HTML позволяет применять стили к элементам с помощью **атрибута `style`**, а также через **встроенные теги `<style>` и внешние CSS-файлы**. Понимание приоритета и специфичности стилей важно для корректного отображения элементов.

**Ключевые моменты:**
- **Атрибут `style`** применяет CSS напрямую к элементу. Эти стили имеют высокий приоритет и могут переопределять правила из `<style>` или внешнего CSS.
- **Встроенные стили в `<style>`** позволяют задавать стили для всех элементов страницы по селекторам. Например, `p { color: teal !important; }` — задаёт цвет текста всех параграфов.
- **`!important`** повышает приоритет правила, делая его почти непреодолимым для других стилей, кроме атрибутов `style` с `!important`.
- **Внешние файлы CSS** (`<link rel="stylesheet" href="styles.css">`) позволяют централизованно управлять стилями сайта.
- **Конфликты стилей** решаются по приоритету:  
  1. Атрибут `style` на элементе  
  2. Встроенные `<style>`  
  3. Внешний CSS  
  4. Браузерные стили по умолчанию  

В примере показано сочетание этих подходов, включая перекрытие цветов параграфов и заголовков, а также стили для ссылок и секций. Это помогает понять, какой цвет и оформление в итоге будет применено к элементу.

### Пример
```javascript
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Атрибут style</title>
  <style>
    p {
      color: teal !important;
    }
    a {
      color: hotpink;
    }
    h2 {
      color: darkblue;
    }

    h2 {
      color: darkorange;
    }
  </style>
  <link rel="stylesheet" href="./styles.css">
</head>
<body>
  <p style="color: red">
    Параграф с красным цветом
  </p>
  <p style="color: green; font-weight: 700">
    Параграф с зеленым жирным текстом
  </p>

  <hr>

  <p>Параграф с бирюзовым текстом</p>
  <a href="#">Ссылка розового цвета</a>

  <hr>

  <h1>Заголовок синего цвета</h1>

  <hr>

  <header>"Шапка" сайта</header>
  <p>Параграф</p>
  <a href="#">Ссылка</a>
  <section>Секция 1</section>
  <section>Секция 2</section>
  <section>Секция 3</section>

  <hr>

  <p style="color: grey !important; font-weight: 700">Какого цвета этот параграф?</p>

  <hr>

  <h2 style="background-color: #141414;">
    А этот заголовок какого цвета?
  </h2>

  <hr>

  <header>
    <nav>
      <ul>
        <li>Пункт 1</li>
        <li>Пункт 2</li>
        <li>Пункт 3</li>
      </ul>
    </nav>
  </header>

  <hr>

  <a href="l2.html">Идти в урок 2</a>
</body>
</html>
```
