# CSS: Логические псевдоклассы :where() и :is()

## Что я узнал

Псевдоклассы `:where()` и `:is()` позволяют писать более гибкие селекторы в CSS. Основная их особенность — управление специфичностью:

- `:where()` **всегда имеет нулевую специфичность**, даже если внутри сложные селекторы. Это делает его идеальным для обнуления стилей и нормализации (например, обнуление отступов у списков или заголовков с классами), без влияния на переопределение стилей более конкретными правилами.
- `:is()` позволяет группировать селекторы и облегчает запись сложных комбинаций без увеличения специфичности.

Главное отличие от классических селекторов (`:nth-child`, `:not`) в том, что они не повышают специфичность кода, что дает большую гибкость при переопределении стилей.

### Пример
```css
/* Нормализация отступов для списков и заголовков */
:where(ul, ol, h1, h2, h3, h4, h5, h6) {
  margin: 0;
  padding: 0;
}

/* Группировка селекторов через :is() */
:is(h1, h2, h3).title {
  font-weight: bold;
  color: navy;
}
```
