# Видео курс по верстке и Scratch

## Что я узнал

Этот пример демонстрирует полноценную структуру веб-страницы с использованием HTML, CSS и Bootstrap для курсов по Scratch. Включены секции: header с кнопками и модальным видео, "О курсе" с иконками и описанием преимуществ, "Курс для тех, кто" с таргетированными блоками, "Программа курса" с аккордеонами для уроков, "Процесс обучения", "Об авторе" и "Купить" с формой, а также footer с социальными ссылками. CSS использует кастомные переменные `--color1` и `--color2`, а также современные свойства, такие как `aspect-ratio`, flexbox для центрирования, hover-эффекты и адаптивные размеры через Bootstrap сетку.  

Также показаны приемы создания интерактивных элементов через data-атрибуты для модальных окон и анимации кнопок. Курс ориентирован на структурированный материал, символическую цену, практические задания и мобильную доступность.
