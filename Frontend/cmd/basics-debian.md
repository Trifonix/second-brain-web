# Основы терминала в Linux (Debian)

## Что я узнал

В Linux терминал является основной точкой взаимодействия с системой через командную строку. В отличие от Windows CMD, здесь используется оболочка (обычно `bash`), поддерживающая богатый набор команд для управления файлами, папками и процессами. Основные концепции и команды:

- **Открытие терминала**: сочетания клавиш зависят от среды рабочего стола (например, `Ctrl + Alt + T`).
- **Навигация по файловой системе**:
  - `pwd` — показывает текущую директорию.
  - `cd YPOK` — перейти в папку YPOK.
  - `ls` — список файлов и папок в текущей директории.
- **Создание и управление файлами и папками**:
  - `mkdir YPOK` — создать папку YPOK.
  - `touch A1.txt` — создать пустой текстовый файл.
  - `echo "текст" > A1.txt` — записать текст в файл.
  - `mv A1.txt A2.txt` — переименовать или переместить файл.
- **Очистка экрана**: `clear`.
- **Цвет текста и форматирование**: через ANSI escape-последовательности (например, `echo -e "\e[32mТекст зелёного цвета\e[0m"`).

Использование этих команд позволяет эффективно работать с файловой системой, создавать и изменять файлы без графического интерфейса, а также управлять выводом текста в терминале.

### Пример
```javascript
// Создание папки и файла с текстом в терминале Debian
mkdir YPOK          // создаем папку
cd YPOK             // переходим в папку
touch A1.txt        // создаем пустой файл
echo "Привет, мир" > A1.txt  // записываем текст в файл
mv A1.txt A2.txt    // переименовываем файл
clear               // очищаем экран
```
