# Основы CMD в Windows 10

## Что я узнал

В Windows 10 есть встроенная командная строка (CMD), позволяющая управлять файлами, папками и выполнять различные системные команды без графического интерфейса. Для быстрого доступа к CMD можно использовать сочетание клавиш **WIN + R** и ввод команды `cmd`. После открытия командной строки доступны следующие базовые операции:

- **color a** — изменить цвет текста на лаймовый.
- **cls** — очистить экран командной строки.
- **md YPOK** — создать новую папку с именем YPOK.
- **cd YPOK** — перейти внутрь папки YPOK.
- **copy con A1.txt** — создать текстовый файл A1.txt и сразу ввести в него текст.
- **CTRL+Z** — завершить ввод текста при использовании команды `copy con`.
- **ren A1.txt A2.txt** — переименовать файл A1.txt в A2.txt.

Эти команды формируют базу для работы с файловой системой через CMD и позволяют автоматизировать рутинные задачи без использования графического интерфейса.

### Пример
```javascript
// Пошаговое использование команд в CMD
WIN + R
// Ввести:
cmd

// В открытой командной строке:
color a        // меняем цвет текста на лаймовый
cls            // очищаем экран
md YPOK        // создаем папку YPOK
cd YPOK        // переходим в папку
copy con A1.txt
// вводим текст, затем CTRL+Z для сохранения
ren A1.txt A2.txt // переименовываем файл
```
