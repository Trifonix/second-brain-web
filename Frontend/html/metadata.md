# HTML метаданные (meta теги)

## Что я узнал

Метаданные — это информация о веб-странице, которая **не отображается напрямую пользователю**, но используется браузерами, поисковыми системами и другими сервисами. Метаданные размещаются внутри тега `<head>` и чаще всего задаются с помощью тега `<meta>`.

Тег `<meta charset="UTF-8">` определяет кодировку символов страницы. Кодировка UTF-8 является стандартной и позволяет корректно отображать большинство символов разных языков.

Метаданные также могут описывать содержимое страницы. Например:
- `description` — краткое описание страницы, которое часто используется поисковыми системами.
- `keywords` — ключевые слова, связанные с содержанием страницы.
- `author` — имя автора страницы или сайта.

Тег `<meta name="viewport">` важен для адаптивного дизайна. Он помогает корректно отображать страницу на мобильных устройствах и управляет масштабированием страницы.

Также можно использовать `http-equiv`. В примере используется `refresh`, который автоматически **перезагружает страницу через определённое количество секунд**. В данном случае значение `content="30"` означает обновление страницы каждые 30 секунд.

Метаданные помогают поисковым системам лучше понимать страницу, а браузерам — правильно её отображать и обрабатывать.

### Пример
```javascript
<!DOCTYPE html>

<html>
    <head>
        
        <title>My webpage</title>

        <meta charset="UTF-8">
        <meta name="description" content="Free HTML tutorial for beginners">
        <meta name="keywords" content="HTML, tutorial, beginners">
        <meta name="author" content="Bro Code">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="refresh" content="30">

    </head>

    <body>
        <h1>Lesson: METADATA</h1>
    </body>
</html>
```
