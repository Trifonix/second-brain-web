# Базовая структура HTML-документа

## Что я узнал

Каждый HTML-документ имеет стандартную структуру, которую браузер ожидает увидеть.  
Эта структура определяет тип документа, язык страницы, кодировку, мета-информацию и основной контент.

Документ начинается с `<!DOCTYPE html>` — это объявление версии HTML (HTML5).  
Оно сообщает браузеру, что используется современный стандарт.

Тег `<html>` — корневой элемент страницы.  
Атрибут `lang` указывает язык документа и важен для доступности и SEO.

Внутри `<html>` есть два основных раздела:

- `<head>` — служебная информация (метаданные, заголовок, подключение стилей и скриптов)
- `<body>` — весь видимый контент страницы

Внутри `<head>` обязательно должны быть:

- `<meta charset="UTF-8">` — кодировка документа (чтобы корректно отображались символы)
- `<meta name="viewport" content="width=device-width, initial-scale=1.0">` — адаптивность на мобильных устройствах
- `<title>` — заголовок вкладки браузера

Важно понимать:
- Без `viewport` сайт будет плохо отображаться на мобильных устройствах.
- Без правильной кодировки возможны проблемы с отображением текста.
- `<main>` должен быть один на странице.
- Структура документа — это фундамент всей верстки.

---

### Пример
```javascript
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Моя страница</title>
</head>
<body>

  <header>
    <h1>Заголовок сайта</h1>
  </header>

  <main>
    <section>
      <h2>Основной контент</h2>
      <p>Текст страницы...</p>
    </section>
  </main>

  <footer>
    <p>© 2026</p>
  </footer>

</body>
</html>
```
