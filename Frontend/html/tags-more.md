# Дополнительные семантические и текстовые теги в HTML

## Что я узнал

HTML предоставляет множество **семантических и текстовых тегов**, которые помогают лучше структурировать информацию, улучшить доступность и SEO, а также добавить интерактивные элементы без использования JavaScript.

**Интерактивные элементы:**
- `<details>` — создаёт раскрывающийся блок контента.
- `<summary>` — заголовок для блока `<details>`, который виден всегда и по клику раскрывает содержимое.
- `<progress>` — индикатор выполнения задачи или загрузки, позволяет визуально показать прогресс выполнения процесса.

**Текстовые семантические элементы:**
- `<abbr>` — обозначает аббревиатуру, атрибут `title` даёт расшифровку.  
- `<address>` — контактная информация автора или организации.  
- `<cite>` — цитирование источника, например название книги или статьи.  
- `<q>` — короткая встроенная цитата, автоматически обрамляется кавычками браузером.

Использование этих тегов делает HTML более **понятным для браузеров, поисковых систем и вспомогательных технологий**, а также улучшает восприятие контента пользователями.

### Пример
```javascript
<h2>Интерактивные блоки</h2>
<details>
    <summary>Подробнее о HTML</summary>
    <p>HTML — это язык разметки, используемый для создания веб-страниц.</p>
</details>

<progress value="70" max="100"></progress>

<h2>Текстовые семантические элементы</h2>
<p>Аббревиатура: <abbr title="HyperText Markup Language">HTML</abbr></p>
<p>Контакты автора: <address>email@example.com</address></p>
<p>Цитата источника: <cite>Bro Code, HTML Guide</cite></p>
<p>Короткая цитата: <q>Учение — свет</q></p>
```
