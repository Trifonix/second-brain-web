# Цвета и стили в HTML (inline CSS)

## Что я узнал

В HTML можно изменять внешний вид элементов с помощью CSS. Один из самых простых способов — использовать **inline CSS**, то есть задавать стили прямо внутри HTML-тега через атрибут `style`.

С помощью свойства `background-color` можно изменить цвет фона элемента. Например, если задать это свойство для тега `<body>`, то изменится фон всей страницы.

Свойство `color` используется для изменения цвета текста. Его можно применять к заголовкам (`<h1>`, `<h2>`, `<h3>`), параграфам (`<p>`) и другим текстовым элементам.

Цвета в CSS можно задавать несколькими способами:
- **Название цвета** — например `red`, `black`.
- **RGB** — формат `rgb(red, green, blue)`, где значения задаются числами от 0 до 255.
- **HEX** — шестнадцатеричный код цвета, который начинается с `#`.

В примере фон всей страницы сделан чёрным (`black`). Заголовки используют разные способы задания цвета: название цвета (`red`), RGB (`rgb(5, 244, 200)`) и HEX (`#28f015`). Параграф имеет тёмный фон и зелёный текст, что создаёт контраст и делает текст легче читаемым.

Inline-стили удобно использовать для быстрых экспериментов или небольших примеров, но в реальных проектах чаще применяют отдельные CSS-файлы.

### Пример
```javascript
<body style="background-color: black;">
    <h1 style="color: red;">Hello!</h1>
    <h2 style="color: rgb(5, 244, 200);">Hello h2</h2>
    <h3 style="color: #28f015">Hello h3</h3>

    <p style="background-color: #151515; color: #28f015;">
        Lorem ipsum dolor sit amet consectetur, adipisicing elit. Dolor accusantium explicabo modi! Quibusdam nesciunt sunt eum eaque, explicabo vel at natus dicta illum quo, reiciendis quas laudantium, consequatur expedita ipsa?
    </p>
</body>
```
