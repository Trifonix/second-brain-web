# Сложная разметка таблиц в HTML

## Что я узнал

HTML таблицы можно делать более **сложными и функциональными**, используя дополнительные атрибуты и элементы. Это позволяет создавать более структурированные данные и даже макеты, хотя использование таблиц для верстки страниц считается устаревшим.

**Объединение ячеек:**
- `rowspan` — объединяет несколько строк в одну ячейку по вертикали.
- `colspan` — объединяет несколько колонок в одну ячейку по горизонтали.  

Эти атрибуты полезны, когда нужно создать заголовки или ячейки, которые охватывают несколько строк или столбцов, например в расписаниях или комплексных таблицах данных.

**Подписи к таблицам:**
- `<caption>` — элемент для добавления названия таблицы. Он обычно отображается над таблицей и помогает понять её содержимое.

**Использование таблиц для макета:**
- В прошлом таблицы часто использовались для позиционирования элементов на странице. Сейчас это устаревшая практика, но знание этого важно для поддержки старых проектов. Таблицы всё ещё могут применяться для отображения **табличных данных**, прайс-листов, расписаний и отчётов.

### Пример
```javascript
<table border="1" width="600">
    <caption>Weekly Schedule</caption>
    <tr>
        <th>Time</th>
        <th>Monday</th>
        <th>Tuesday</th>
        <th>Wednesday</th>
    </tr>
    <tr>
        <td>Morning</td>
        <td rowspan="2">Work</td>
        <td>Gym</td>
        <td>Meeting</td>
    </tr>
    <tr>
        <td>Afternoon</td>
        <td>Lunch</td>
        <td>Work</td>
    </tr>
    <tr>
        <td>Evening</td>
        <td colspan="2">Relax / Study</td>
        <td>Family time</td>
    </tr>
</table>
```
