# Интерактивность и глобальные атрибуты в HTML

## Что я узнал

HTML предоставляет ряд **глобальных атрибутов**, которые делают страницу более интерактивной и удобной для пользователей **без необходимости писать сложный JavaScript**.  

- **`contenteditable`** — делает элемент редактируемым пользователем прямо на странице. Например, текст в `<p contenteditable>` можно менять прямо в браузере.  
- **`tabindex`** — управляет порядком фокусировки элементов при нажатии клавиши Tab. Позволяет улучшить навигацию по форме или странице с клавиатуры.  
- **`spellcheck`** — включает или отключает проверку орфографии в полях ввода и текстовых блоках. Полезно для текстовых редакторов и форм.  
- **`data-*` атрибуты** — позволяют хранить **пользовательские данные внутри HTML-тегов**, которые потом можно использовать в JavaScript для динамического поведения страницы. Например, `<div data-id="123"></div>`.

Использование этих атрибутов делает страницы более **динамичными, доступными и интерактивными** без значительных затрат на скрипты.

### Пример
```javascript
<h2 contenteditable="true">Редактируемый заголовок</h2>

<p tabindex="1">Элемент с первым порядком фокуса</p>
<p tabindex="2">Элемент со вторым порядком фокуса</p>

<textarea spellcheck="true">Проверка орфографии включена здесь</textarea>

<div data-user-id="42" data-role="admin">
    Пользовательские данные сохранены в HTML-атрибутах
</div>
```
