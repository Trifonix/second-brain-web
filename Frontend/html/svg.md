# Графика и SVG в HTML

## Что я узнал

HTML позволяет работать не только с растровыми изображениями через тег `<img>`, но и создавать **векторную графику прямо в коде страницы** с помощью SVG (Scalable Vector Graphics). SVG сохраняет чёткость при любом масштабе и позволяет легко анимировать и стилизовать графику с помощью CSS и JavaScript.

Основные элементы SVG:
- `<svg>` — контейнер для всей векторной графики.
- `<path>` — задаёт произвольные линии и формы с помощью атрибута `d`, где описываются координаты и кривые.
- `<g>` — группа элементов SVG, позволяет объединять несколько фигур для применения общих стилей или трансформаций.
- `<polygon>` — рисует многоугольник по набору координат.

SVG подходит для создания **иконок, логотипов, диаграмм и интерактивной графики**, и позволяет гибко управлять стилями и анимацией без использования сторонних изображений.

### Пример
```javascript
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
    <!-- Группа для общей трансформации -->
    <g fill="none" stroke="black" stroke-width="2">
        <!-- Произвольная линия -->
        <path d="M10 80 C 40 10, 65 10, 95 80 S 150 150, 180 80"/>
        <!-- Треугольник -->
        <polygon points="50,150 100,50 150,150" fill="lightblue" stroke="blue"/>
    </g>
</svg>
```
