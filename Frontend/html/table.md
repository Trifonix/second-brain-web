# HTML таблицы: создание и стилизация

## Что я узнал

HTML таблицы используются для **структурирования данных в виде строк и столбцов**. Основные элементы таблицы:

- `<table>` — контейнер для всей таблицы.
- `<tr>` (table row) — строка таблицы.
- `<th>` (table header) — заголовочная ячейка, обычно выделяется жирным и по центру.
- `<td>` (table data) — обычная ячейка с данными.

Атрибуты позволяют настраивать внешний вид таблицы:
- `bgcolor` задаёт цвет фона для всей таблицы, строки или ячейки.
- `width` определяет ширину таблицы или конкретной ячейки.
- `align` задаёт выравнивание содержимого (`left`, `center`, `right`).

В примере таблица показывает **часы работы магазина**. Заголовки дней недели размещены в `<th>`, а данные с часами работы в `<td>`. Для удобочитаемости строки имеют разные цвета фона: заголовочная строка — серый, строки с данными — светло-серый. Значения `Closed` показывают, что магазин не работает в выходные.

Таблицы полезны для расписаний, прайс-листов, статистических данных и любой информации, где важно расположение данных в сетке.

### Пример
```javascript
<h3>Store Hours</h3>

<table bgcolor="black" width="700">
    <tr bgcolor="grey">
        <th width="100">Monday</th>
        <th width="100">Tuesday</th>
        <th width="100">Wednesday</th>
        <th width="100">Thursday</th>
        <th width="100">Friday</th>
        <th width="100">Saturday</th>
        <th width="100">Sunday</th>
    </tr>
    <tr bgcolor="lightgray" align="center">
        <td>7-18</td>
        <td>7-18</td>
        <td>7-18</td>
        <td>7-18</td>
        <td>7-18</td>
        <td>Closed</td>
        <td>Closed</td>
    </tr>
</table>
```
