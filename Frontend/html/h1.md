# Базовая структура HTML-документа

## Что я узнал

HTML-документ имеет стандартную базовую структуру, которая сообщает браузеру, как интерпретировать страницу.  
В начале документа указывается `<!DOCTYPE html>`, что говорит браузеру использовать современный стандарт HTML5.

Тег `<html>` является корневым элементом всей страницы. Атрибут `lang="en"` указывает язык документа, что полезно для браузеров, поисковых систем и технологий доступности.

Секция `<head>` содержит метаданные страницы — информацию, которая не отображается напрямую на странице, но используется браузером. Например:
- `<meta charset="UTF-8">` задаёт кодировку символов, позволяя корректно отображать текст.
- `<meta name="viewport">` управляет масштабированием страницы на мобильных устройствах.
- `<title>` задаёт заголовок страницы, который отображается на вкладке браузера.

Секция `<body>` содержит весь видимый контент страницы. В данном примере используется атрибут `style`, который задаёт фоновый цвет страницы (`background-color: aquamarine`). Это пример **inline CSS** — когда стили прописываются прямо внутри HTML-тега.

Тег `<h1>` используется для основного заголовка страницы и обычно обозначает главный смысловой заголовок документа.

Таким образом, пример показывает минимальную рабочую HTML-страницу с базовой структурой, метаданными и простым содержимым.

### Пример
```javascript
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body style="background-color: aquamarine;">
    <h1>This is Advertisement</h1>
</body>
</html>
```
