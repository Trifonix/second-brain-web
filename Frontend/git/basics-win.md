# Основы GIT в Windows 10

## Что я узнал

GIT — это система контроля версий, которая позволяет отслеживать изменения в коде, работать с ветками и синхронизировать локальные и удалённые репозитории. В Windows 10 её использование обычно осуществляется через **Git Bash** или интегрированные терминалы в редакторах, таких как Visual Studio Code.

Основные шаги и команды:

1. **Установка и проверка GIT**
   ```text
   git --version
```

Показывает установленную версию GIT.

2. **Настройка пользователя**

   ```text
   git config --global user.name "Ваше Имя"
   git config --global user.email "email@example.com"
   ```

   Настраивает имя и email для коммитов.

3. **Инициализация репозитория**

   ```text
   git init
   ```

   Создаёт локальный репозиторий в текущей папке.

4. **Добавление файлов и создание коммита**

   ```text
   git add .
   git commit -m "Первый коммит"
   ```

   `git add .` — добавляет все изменения в индекс, `git commit` фиксирует их с сообщением.

5. **Работа с удалённым репозиторием**

   ```text
   git remote add origin https://github.com/username/repo.git
   git push -u origin main
   ```

   Подключение удалённого репозитория и отправка изменений.

6. **Просмотр статуса и истории**

   ```text
   git status
   git log --oneline
   ```

   Проверка текущего состояния и истории коммитов.

7. **Работа с ветками**

   ```text
   git branch new-feature
   git checkout new-feature
   git merge new-feature
   ```

   Создание, переключение и слияние веток.

Использование GIT в Windows 10 позволяет удобно отслеживать изменения, работать в команде и откатывать нежелательные изменения в коде.

### Пример

```javascript
// Инициализация нового репозитория и коммит
git init
git add .
git commit -m "Инициализация проекта"
```
