# JavaScript: Оператор нулевого слияния и логические операторы присваивания

## Что я узнал

В JavaScript есть несколько операторов, которые помогают сокращать код и управлять значениями переменных:

1. **Оператор нулевого слияния `??`**  
   - Возвращает правое значение, только если левое **`null` или `undefined`**.  
   - Отличается от логического ИЛИ `||`, который возвращает правое значение, если левое **любое falsy** (`0`, `''`, `false`, `null`, `undefined`, `NaN`).  
   - Полезен для задания значений по умолчанию без риска перезаписать допустимые falsy значения вроде 0 или пустой строки.

2. **Логические операторы присваивания**  
   - `||=` — присваивает значение только если переменная falsy.  
   - `&&=` — присваивает значение только если переменная truthy.  
   - `??=` — присваивает значение только если переменная `null` или `undefined`.  
   - Эти операторы сокращают стандартную форму вида `if (!a) a = value` или `a = a ?? value`.

Эти правила часто встречаются в практике, но могут отсутствовать в базовых конспектах по операторам.

### Пример
```javascript
// Оператор нулевого слияния
let a = null;
let b = a ?? 10; // 10, т.к. a null
console.log(b);

let c = 0;
let d = c ?? 5; // 0, т.к. c не null/undefined
console.log(d);

let e = 0;
let f = e || 5; // 5, т.к. e falsy
console.log(f);

// Логические операторы присваивания
let user;
user ||= "DefaultUser"; // если falsy, присвоить
console.log(user); // "DefaultUser"

let admin = true;
admin &&= false; // если truthy, присвоить новое значение
console.log(admin); // false

let count;
count ??= 42; // если null/undefined, присвоить
console.log(count); // 42
```
