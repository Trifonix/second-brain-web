# Тернарный оператор

## Что я узнал

Тернарный оператор — это краткая форма условного выражения `if...else`, которая позволяет записать проверку условия и выбор одного из двух значений в одну строку. Он используется тогда, когда нужно присвоить переменной значение в зависимости от условия. Синтаксис тернарного оператора выглядит так: `условие ? значение_если_true : значение_если_false`.

Главное отличие от обычного `if` в том, что тернарный оператор — это **выражение**, которое возвращает значение. Поэтому его удобно использовать прямо внутри присваивания, аргументов функции или других выражений. Такой способ делает код компактнее, но важно не злоупотреблять им: если логика становится сложной, обычный `if...else` будет читаться лучше.

Тернарные операторы часто применяются в UI-логике, при вычислении значений переменных, а также в шаблонах и JSX.

### Пример
```javascript
let a = '1';
let value = 5;

// обычный if...else
if (a > 0) {
  value = a;
} else {
  value += 1;
}

// тернарный оператор
// условие ? значение если true : значение если false
value = a === 1 ? 8 : 42;
```
