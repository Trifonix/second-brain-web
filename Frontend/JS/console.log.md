# console.log()

## Что я узнал

`console.log()` — метод объекта `console` в JavaScript для вывода данных в консоль. Принимает любые аргументы, форматирует объекты/массивы, идеален для отладки. Не прерывает код.

**Особенности:**
- Множественные аргументы: `console.log('Имя:', name)`
- Интерполяция: `console.log(`Город: ${city}`)`
- Работает в браузере и Node.js

## Пример

```javascript
const user = { name: 'Алексей', age: 28 };
console.log('Пользователь:', user);
console.log(`Привет из Севастополя!`);
```

Вывод:

```
Пользователь: {name: "Алексей", age: 28}
Привет из Севастополя!
```
