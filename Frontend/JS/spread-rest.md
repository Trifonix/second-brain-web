# JavaScript: Оператор Spread и Rest

## Что я узнал

Оператор **spread (`...`)** и **rest (`...`)** — это два разных применения одного синтаксического конструкта, которые позволяют удобно работать с массивами, объектами и функциями.  

- **Spread** используется для **разворачивания** элементов массива или свойств объекта в новый массив/объект или в аргументы функции.  
- **Rest** используется для **сбора** оставшихся элементов или аргументов функции в массив. В теории чаще упоминаются только rest-параметры, но на практике spread встречается гораздо чаще, например, при клонировании данных.

Эти операторы позволяют писать более чистый и лаконичный код, избегая явных циклов и `concat` при работе с коллекциями.

### Пример
```javascript
// Spread для массивов
const arr1 = [1, 2, 3];
const arr2 = [4, 5];
const combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4, 5]

// Spread для объектов
const obj1 = { a: 1, b: 2 };
const obj2 = { b: 3, c: 4 };
const merged = { ...obj1, ...obj2 };
console.log(merged); // { a: 1, b: 3, c: 4 }

// Rest-параметры в функции
function sumAll(...numbers) {
  return numbers.reduce((acc, n) => acc + n, 0);
}
console.log(sumAll(1, 2, 3, 4)); // 10

// Spread для передачи аргументов
const nums = [5, 10, 15];
console.log(Math.max(...nums)); // 15
```
