# Продвинутый JavaScript и работа с DOM

## Что я узнал

JavaScript позволяет не только выполнять вычисления и работать с объектами, но и **управлять содержимым страницы через DOM (Document Object Model)**. DOM — это объектная модель HTML-документа, которая позволяет находить элементы, изменять их свойства, добавлять обработчики событий и динамически менять внешний вид интерфейса.

В более продвинутом использовании JavaScript активно применяется **обработка событий**, **манипуляция классами** и **изоляция кода**.

Одним из основных инструментов является метод `addEventListener`. Он позволяет подписаться на различные события:

- клик мыши
- наведение
- ввод текста
- прокрутку страницы
- нажатия клавиш

Это основа для создания интерактивных элементов интерфейса, например:

- модальных окон
- выпадающих меню
- кнопок открытия и закрытия
- переключателей

Часто вместе с обработкой событий используется объект `classList`. Он позволяет **управлять CSS-классами элемента прямо из JavaScript**.

Основные методы:

- `classList.add()` — добавить класс
- `classList.remove()` — удалить класс
- `classList.toggle()` — переключить класс
- `classList.contains()` — проверить наличие класса

Метод `classList.toggle()` особенно удобен для **динамического изменения интерфейса**. Например, при клике можно:

- открыть меню
- показать модальное окно
- переключить тему
- активировать анимацию

Еще один важный инструмент — **IIFE (Immediately Invoked Function Expression)**.

Это **самовызывающаяся функция**, которая выполняется сразу после объявления.

Она используется для:

- изоляции переменных
- предотвращения конфликтов в глобальной области видимости
- организации модульного кода

Без изоляции переменные могут попадать в глобальный объект `window`, что приводит к конфликтам между скриптами.

IIFE решает эту проблему, создавая **собственную область видимости**.

Таким образом, современные практики JavaScript включают:

- работу с DOM
- реакцию на пользовательские события
- динамическое изменение классов
- организацию безопасной структуры кода

Эти техники лежат в основе большинства интерактивных интерфейсов на веб-сайтах.

### Пример
```javascript
// IIFE — изоляция кода
(function () {

  // Находим элементы
  const button = document.querySelector('.menu-button');
  const menu = document.querySelector('.menu');
  const modalButton = document.querySelector('.open-modal');
  const modal = document.querySelector('.modal');

  // Открытие / закрытие меню
  button.addEventListener('click', () => {
    menu.classList.toggle('is-active');
  });

  // Открытие модального окна
  modalButton.addEventListener('click', () => {
    modal.classList.add('is-active');
  });

  // Закрытие модального окна
  modal.addEventListener('click', (event) => {
    if (event.target === modal) {
      modal.classList.remove('is-active');
    }
  });

})();
```
