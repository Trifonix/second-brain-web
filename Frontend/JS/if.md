# Условные операторы (JavaScript)

## Что я узнал

Условные операторы позволяют выполнять код в зависимости от результата логического выражения.

Любое значение в JavaScript может быть приведено к true или false.

Операторы сравнения:

- `==` — нестрогое сравнение, приводит типы  
- `===` — строгое сравнение значения и типа  
- `!=` — нестрогое неравенство  
- `!==` — строгое неравенство  
- `>`, `<`, `>=`, `<=` — операторы сравнения  

Логические преобразования:

- `!value` — логическое отрицание  
- `!!value` — приведение к boolean  

Falsy значения:

0  
''  
null  
undefined  
NaN  

Все остальные значения являются truthy.

---

## Пример

```javascript
let a = 2;
let b = '2';

a == b;   // true
a === b;  // false

a > b;
a < b;
a >= b;
a <= b;

a != b;
a !== b;

let value = !!a;

if (!value) {
  console.log('i am here');
} else {
  console.log('im here');
}
```
