# Objects (JavaScript)

## Что такое объект

Объект — это структура данных, которая хранит значения в формате  
**ключ → значение**.

Объекты используются для хранения связанных данных в одной сущности.

Пример: описание пользователя, товара, настроек и т.д.

---

## Создание объекта

```js
let obj = {
  firstName: 'Smith',
  age: 25
};
```

Здесь:

- `firstName` и `age` — свойства (ключи)
- `'Smith'` и `25` — значения

---

## Доступ к свойствам

### Через точку

```js
console.log(obj.age);
```

### Через квадратные скобки

```js
console.log(obj['age']);
```

Оба способа вернут:

25

---

## Когда использовать квадратные скобки

Если имя свойства хранится в переменной:

```js
let key = 'age';
console.log(obj[key]); // 25
```

---

## Изменение значения

```js
obj.age = 30;
```

---

## Добавление нового свойства

```js
obj.city = 'London';
```

---

## Использование с DOM

Можно вывести значение свойства на страницу:

```js
document.body.innerHTML = obj['age'];
```

В браузере отобразится:

25

---

## Важно помнить

- Объект — это ссылочный тип данных.
- Ключи объекта по сути являются строками.
- Если обратиться к несуществующему свойству — вернётся `undefined`.

Пример:

```js
console.log(obj.salary); // undefined
```
