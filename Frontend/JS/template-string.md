# Template String (JavaScript)

## Что это

Template string (шаблонная строка) — это строка в JavaScript, которая записывается в обратных кавычках `` ` ``  
и позволяет вставлять переменные и выражения внутрь строки через `${}`.

---

## Синтаксис

```js
const name = "Alex";

const message = `Привет, ${name}!`;
console.log(message);
```

Результат:

Привет, Alex!

---

## Вставка выражений

Внутрь `${}` можно вставлять не только переменные, но и выражения:

```js
const a = 5;
const b = 3;

console.log(`Сумма: ${a + b}`);
```

Результат:

Сумма: 8

---

## Многострочные строки

Template string поддерживает перенос строки без использования `\n`:

```js
const text = `
Первая строка
Вторая строка
Третья строка
`;

console.log(text);
```

---

## Использование с HTML

Template string удобно использовать для генерации HTML-разметки:

```js
const title = "Заголовок";
const content = "Это текст статьи";

const html = `
  <h1>${title}</h1>
  <p>${content}</p>
`;

document.body.innerHTML = html;
```

Если внутри шаблона есть HTML-код, он будет отображён на странице как разметка.

---

## Важно помнить

- Используются обратные кавычки `` ` ``, а не `' '` или `" "`.
- Значения вставляются только через `${}`.
- При использовании `innerHTML` нужно быть осторожным — можно вставить небезопасный код.
