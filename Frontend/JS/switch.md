# switch-case

## Что я узнал

`switch` — это управляющая конструкция в JavaScript, которая используется для выбора одного из нескольких вариантов выполнения кода на основе значения выражения. Она удобна в ситуациях, когда нужно сравнить одну переменную с несколькими возможными значениями. Это делает код более читаемым по сравнению с длинной цепочкой `if...else if`.

Работа `switch` начинается с вычисления выражения в скобках. Затем это значение последовательно сравнивается с каждым `case`. Если совпадение найдено, выполняется код внутри этого блока.

После выполнения блока обычно используется оператор `break`. Он завершает выполнение `switch` и предотвращает переход к следующему `case`. Если `break` не поставить, выполнение продолжится дальше — это поведение называется **fall-through**.

Блок `default` выполняется, если ни один из `case` не совпал со значением выражения. Он играет роль аналогичную `else` в условных конструкциях.

`switch` особенно полезен, когда нужно выбрать действие на основе конкретного значения переменной (например, тип события, статус, цвет, тип команды и т.д.).

### Пример
```javascript
let color = 'orange';
let value;

switch (color) {
  case 'yellow':
    value = 1;
    break;

  case 'red':
    value = 2;
    break;

  case 'orange':
    value = 100;
    break;

  default:
    value = 0;
}
```
